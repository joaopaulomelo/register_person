## About Project

API for registering and simple consultation of people.

## Tools
 
- **[Tests with phpunit](https://phpunit.de/)** 
- **[Laravel Framework](https://laravel.com/)** 

## Run Project

- composer install
- php artisan migrate --seed
- php artisan key:generate
- php artisan serve

## Run Tests

- php artisan test
